# xone-dkms Debian Package

This directory contains Debian packaging files for **xone**, a Linux kernel driver for Xbox One and Xbox Series X|S accessories.

## About xone

xone is a modern Linux kernel driver that serves as a replacement for the legacy xpad driver. It implements Microsoft's Game Input Protocol (GIP) and provides support for:

- Xbox One and Xbox Series X|S controllers (wired)
- Xbox Wireless Adapter for USB connectivity
- Third-party Xbox-compatible controllers
- Force feedback and rumble
- Battery status reporting
- LED control
- Audio support

This package is based on the actively maintained fork at https://github.com/dlundqvist/xone
